事前準備網頁

  • 個人簡介

    彭彭老師您好：

    我是黃偉豪，來自桃園市，目前就讀中山大學資訊管理研究所二年級。我大學時並非唸資訊相關科系，畢業後，我在家附近的一間公司做著不需要具備一定程度專業的工作，那時候我問自己：我一生就打算這樣過了嗎？領著餓不死的薪水住在家裡，多年後朋友們都成長了，可能只剩我還在原地踏步。於是我存了一點錢，放下工作，毅然決然進入研究所就讀，一方面是彌補自己大學時的不認真，我更加珍惜這兩年的求學生活；一方面是能夠得到資訊碩士畢業證書。對於未來我充滿期待，但知道自己能力還有所不足，因此希望能夠加入老師的扶持計畫訓練營。

  • 為什麼想成為工程師

    我認為工程師是一份適合我的工作，我的個性比較不善於與人互動，如果是需要時常與不同人相處、互動的一些工作，對我來說是一件有壓力的事情。寫程式是一件需要專注思考的事情，對我來說，不斷的反覆嘗試、修正錯誤，最後程式成功執行的那一刻，是非常有成就感的，雖然任何工作都避不開與人相處，不過當我與同學們互相討論如何完成小組的專案時，這個過程我是非常期待的，期待著我們能夠討論出一個方法並且成功執行。

    碩一時，修了一門「資料庫系統」，學習如何設計、操作 web-based 的資料庫應用系統，那時候對於網頁程式語言還沒什麼概念，而期中就要架出一個網站，於是我上網搜尋相關的教學資源，那也是我第一次認識到彭彭老師。最後做出來的網站可能沒有很完善，但自己親手完成時，更能深刻體會網站跟資料庫之間的配合運作，那時我對於後端程式語言如何連線資料庫、執行 SQL 語法以及架設網站，花了非常多時間去搜尋、學習。完成專案時，覺得自己好像有那麼一點點像工程師，但對於網站的整體運作概念還是不太熟悉，因此希望能透過這個計畫，紮穩前端、後端、資料庫的基礎，並且朝向後端工程師這個目標邁進。

  • 為了成為工程師，曾經做過什麼努力？

    年紀越大，做出的改變成本與風險就越高。趁著我還有一點點時間與機會改變，我選擇放手一搏，努力準備資訊研究所考試，在準備考試期間第一次真正了解到程式語言的邏輯運作，實作了一些資料結構演算法，而我也成功正取進入中山大學資管所。

    但研究所不會教你怎麼寫程式，我必須另外找教學資源學習，在看了許多網路教學後，發現彭彭老師的教學方式最適合我，於是開始看了老師的 Python 教學，我在學校的電子商務時報實習，每個月初我要彙整記者資料給助理報帳，於是我寫了一支 Python 的爬蟲協助我完成這項工作；在學完基本的 html、css 後，發現 Pug 跟 Sass 的縮排寫法跟 Python 很像，也想嘗試這種寫法，為了編譯 Pug 跟 Sass，於是看了老師的 webpack 教學，做了一個自己的簡歷網頁；之前有跟著其他教學使用 Python Flask 與 SQLite 做一個簡易的留言板，不過學習的效果不太好，所以很開心老師開了 Flask 的教學。

    累積了一些上課做的專案與小作品後，我有試著去投遞實習，但結果沒有很好，也知道自己能力的確還不夠，希望藉著這項計畫我能擁有完整的作品，可以端出一盤完整的菜給面試官看。

  • 對於工程師的職業想像是什麼？

    每個職業都有辛苦的地方，之前去面試實習時，面試官有提到他以前從早上 9 點到晚上 9 點都在 coding，不過我認為工程師的工作內容相對單純，是個需要團隊合作的工作，要不斷的與成員溝通、討論，而且客戶的需求可能會改來改去，作為一名工程師，可能需要有很好的耐心以及脾氣，不論是面對客戶還是同事。

    另外工程師必須保持好奇心以及學習新東西的興趣，因為不斷會有新的技術出現，對我來說工程師這個職業就是不斷學習並且解決問題。

  • 如果參與這個計畫，會怎麼安排學習時間？

    我預計今年 7 月底畢業，目前大部分的學分都已經修完，主要剩下論文撰寫，除此之外，沒課的時間應該都會拿來做未來畢業就業的準備，如果參與這個計劃，對於老師說明的每周至少 40 小時的學習，我認為是沒問題的。

    時間安排主要就是跟著老師每週安排的進度、提供每日進度報告，讓自己每天都有在寫 code，維持手感，也希望經由跟老師的討論發現自己寫程式的一些缺點與增進問題表達能力。

  • 如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

    寫程式遇到不知道該怎麼做的時候，我通常都會 Google 一下，就會有許多網友告訴你要怎麼處理，用內建函式或是其他套件等各種方式 ...。如果參與這個計劃，我相信許多問題應該 Google 就有答案，我預期會碰到的困難就是如何在這大量資訊中，過濾出自己要的答案，因為每個人都有不同的邏輯跟寫法，我認為自己必須先一定程度的消化這些內容，然後再與老師討論，並且反覆練習，最後再以自己的邏輯思維呈現。

  • 想要對我們說的事情？

    藉著這份事前準備網頁，只想誠摯地對彭彭老師說聲謝謝，彭彭的課程 Youtube 頻道幫助我非常多，未來若自己有能力時，也希望能夠分享自己所擁有的。
